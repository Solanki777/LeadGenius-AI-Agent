# LeadGenius-AI-Agent

AI-powered automation system that extracts data from visiting cards and stores structured business leads automatically.

## Features

- Upload visiting card image
- OCR text extraction
- AI-based data structuring
- Duplicate detection

## Workflow

1. User uploads visiting card
2. OCR extracts text
3. AI structures data
4. System checks duplicates
5. Stores lead in Google Sheets

## Technologies Used

- n8n automation
- Google Gemini AI
- OCR API
- Google Sheets API
- HTML form

## Project Structure

workflow.json → n8n workflow
upload.html → image upload interface
README.md → documentation
